# Todo Application

A simple and responsive **Todo Application** that allows users to manage their daily tasks efficiently.
Users can add, complete, filter, and delete tasks, helping them stay organized and productive.

## Live Preview

🔗 https://torianykhanna.github.io/todo-application/

> Make sure the link works in **incognito mode** so anyone can access it.

## Technologies Used

* React
* TypeScript
* JavaScript (ES6+)
* SCSS
* Vite
* React Hooks
* REST API
* Responsive Design

## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/torianykhanna/todo-application.git
cd todo-application
```

### 2. Install dependencies

```
npm install
```

### 3. Run the project locally

```
npm run dev
```

The project will run on:

```
http://localhost:5173
```

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Filter tasks (All / Active / Completed)
* Responsive layout
* Persistent data using API
